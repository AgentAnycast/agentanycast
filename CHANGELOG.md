# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.7.0] - 2026-03-21

### Added

- Claude Agent SDK adapter example
- AWS Strands adapter example
- FAQ and troubleshooting guide
- Cross-repo version coordination guide

## [0.6.0] - 2026-03-20

### Added

- TypeScript SDK reference documentation
- Integration guides for Dify, n8n, and Langflow
- Streaming documentation with chunked artifact delivery patterns

## [0.5.0] - 2026-03-19

### Added

- Google ADK adapter example
- OpenAI Agents SDK adapter example
- MCP setup and configuration guide
- ANP (Agent Network Protocol) interop documentation

## [0.4.0] - 2026-03-19

### Added

- CrewAI adapter example
- LangGraph adapter example
- Framework integration guides

## [0.3.0] - 2026-03-18

### Added

- Python SDK reference documentation
- Deployment guide for production relay, HTTP bridge, and MCP server
- Protocol specification covering wire format, task lifecycle, and gRPC services

## [0.2.0] - 2026-03-18

### Added

- Getting started guide with installation and first-agent walkthrough
- Architecture documentation covering sidecar model, security, and NAT traversal

## [0.1.0] - 2026-03-17

### Added

- Project README with architecture overview, quick start guide, and feature matrix
- Contributing guidelines with multi-repo workflow and coding standards
- Code of Conduct (Contributor Covenant v2.1)
- Security policy with vulnerability reporting process
- CLA for FSL-licensed components
- Project logo and branding
- GitHub Issue and PR templates
